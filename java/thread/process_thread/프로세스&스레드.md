# Process & Thread

## 멀티태스킹

하나의 CPU가 여러 개의 작업을 빠른 속도로 번갈아 가면서 수행하는 것

cpu는 1초에 수십억번의 연산이 가능하며, 매우 빠르게 여러 프로그램들의 코드를 번갈아가면서 빠르게 실행하여 프로그램이 동시에 동작 하는 것처럼 보이게 한다. (**시분할 기법**)

CPU가 어떤 프로그램을 얼만큼 실행할지는 운영체제가 결정하며 이러한 것을 스케쥴링이라고 한다.
단순히 시간으로만 작업을 분할하지 않고, 다양한 우선순위와 최적화 기법을 사용하여 스케쥴링한다.

## 멀티프로세싱

둘 이상의 프로세서를 사용하여 여러 작업을 동시에 처리하는 것을 말한다.

멀티 프로세싱은 장비의 관점이고 멅티 태스킹은 운영체제 소프트웨어워 관점이다.


프로세스는 프로그램을 실행하였을 면 생성되는 것.

프로세스는 각각 독립적인 메모리 영역을 가진다.
