# 배열

배열에는 정적 배열 (Array)와 동적 배열 (Vector)가 존재한다.

---

## 정적 배열 (Array)

연속된 메모리 공간에 위치한 같은 타입의 요소들의 모음이다.

배열을 생성할때는 연속된 메모리 공간을 크기를 설정해주어야 합니다.

인덱스를 기반으로 랜덤접근이 가능하며 중복을 허용합니다.

연결 리스트(Linked List)의 경우는 랜덤 접근이 불가능하다.

배열의 경우 연속된 메모리 공간을 사용하기 때문에 중간에 요소가 삭제되는 경우 빈 메모리 공간이 발생하게 되어 배열의 연속적인 특징이 깨지게 된다.

따라서 빈 공간을 채워주기 위해서 삭제된 요소의 뒤에 위치하고 있는 모든 요소들을 앞으로 한 칸씩 땡겨주게 됨으로써 삭제시 시간복잡도는 O(n)이 된다.

삽입 또한 배열의 중간에 새로운 요소가 추가되는 경우 요소가 추가 될 위치부터 뒤에 존재하는 모든 요소를 한칸씩 이동해야 한 후 새로운 요소를 추가해야한다.

이 때문에 삽입 역시 시간복잡도는 O(N)이다.

배열은 연속된 메모리 공간에 저장되어있기 때문에 물리적인 저장위치와 논리적인 저장위치가 동일하다. 

이에 따라서 인덱스를 이용해서 저장된 데이터에 바로 접근하는 Random Access가 가능하다.

### 시간 복잡도 & 공간 복잡도

조회 : O(1)
삽입 : O(N)
삭제 : O(N)

### Java에서의 배열

자바는 배열과 같은 데이터를 효과적으로 관리하기 위해 List를 제공한다. 
    - 배열도 제공한다. ex) int[] array = new int[10];

ArrayList의 경우 내부적으로 용량을 기본값 10을 가진다. 

자바에서는 정적 배열을 유연하게 사용할 수 있도록 배열의 메모리가 가득 차게 되는 경우 배열의 크기를 자동으로 늘려주는 동적 배열 ArrayList를 제공한다.
