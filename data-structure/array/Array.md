# 배열

배열에는 정적 배열 (Array)와 동적 배열 (Vector)가 존재한다.

---

## 정적 배열 (Array)

연속된 메모리 공간에 위치한 같은 타입의 요소들의 모음이다.

배열을 생성할때는 연속된 메모리 공간을 크기를 설정해주어야 합니다.

인덱스를 기반으로 랜덤접근이 가능하며 중복을 허용합니다.

연결 리스트(Linked List)의 경우는 랜덤 접근이 불가능하다.

---

## 궁금점

Java에서는 배열의 크기를 지정하지 않으면 배열을 아예 생성할 수 없나? 기본값으로 생성해주는 기능은 없나?

배열의 메모리 공간이 가득 차게 되는 경우 Java는 배열의 크기는 얼마나 키울까?
