# OPUS4.5

같은 작업 대비 opus4.5는 sonnet4.5에 비해서 더 적은 토큰 사용량으로 더 나은 결과물을 만들어낸다. 

다만 모델 별 사용료가 opus가 더 비싸기 때문에 plan을 사용할 때 opus를 사용하면 더 빨리 토큰이나 세션 사용량이 줄어든다. 


opus4.5는 여러 개의 서브 에이전트를 관리하고 조율하는 능력이 뛰어나다. 

독립적으로 수행되는 에에전트 간의 정보 흐름을 효과적으로 조율하여 협력하도록 한다. 

에이전트간의 순서를 맞춰 실행시키고 정보를 교환한다. 

복잡한 작업(시장 분석, 의료 연구, 코드 리펙토링)은 한 AI가 한번에 처리하기 어려운 여러 세부 작업들로 나뉘는데, 

OPUS4.5는 이런 작업들을 여러 에이전트에 분산시키면서도 효과적으로 조율 할 수 있다.
