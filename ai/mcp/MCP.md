## MCP (Model Context Protocol)

앤트로픽이 개발한 오픈소스 AI 프로토콜

AI 모델과 외부 도구 및 데이터 소스 간의 표준화된 연결을 통해 AI가 컨텍스트를 이해하고 복잡한 작업을 수행하도록 돕습니다. 

MCP는 AI 에이전트와 다양한 서비스를 연결하는 과정을 간소화하여, 사용자가 AI 기능을 더욱 쉽게 활용하고 확장할 수 있게 합니다. 

MCP는 사용자의 자연어 요청의 의도를 파악하고, 그 의도를 달성하기 위해 자신의 판단에 따라 여러 API를 포함한 다양한 도구를 알아서 사용한다.

---

claude와 소통하는 MCP 스크립트는 딱 3가지만 기억하면 된다

1. 입력 읽기 (Read) : Claude가 주는 정보 (어떤 도구를, 어떤 파라미터로 실행할지 등)를 표준 입력(stdin)으로 받습니다.
2. 행동하기 (Act) : MCP의 고유 임무를 수행합니다.
3. 출력하기 (Pring) : 행동의 결과를 정해진 JSON 형식으로 표준 출력(stdout), 즉 print 함수를 사용해 Claude에게 돌려줍니다.

Claude는 우리가 만든 스크립트를 실행하고, 이 print된 결과물을 읽어서 자신의 다음 행동을 결정합니다.
